# restify Boilerplate

restify的样板项目。

## 安装和运行

clone项目，并使用自己希望的目录名称：

```
git clone https://github.com/MarshalW/restify-boilerplate.git your_folder_name

```

安装所需的库：

```
npm install
```

启动服务：

```
npm start
```

然后，浏览器打开：http://localhost:3000/echo/hello

看到如下结果即可：

```
{"name":"hello"}
```
